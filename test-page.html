NavSphere GitHub

📋 安装指南

构建扩展
npm install npm run build
加载到 Chrome

访问 chrome://extensions/，开启开发者模式，点击"加载已解压的扩展程序"，选择 dist 文件夹

配置实例

点击扩展图标，在设置页面添加您的 NavSphere 实例 URL

开始测试

在这个页面尝试右键菜单或快捷键功能

🔧 调试工具

如果遇到问题，可以使用以下调试方法：

  • 检查扩展状态：访问 chrome://extensions/
  • 查看后台日志：点击"检查视图：service worker"
  • 调试弹窗：右键扩展图标选择"检查弹窗"
  • 控制台日志：按 F12 查看当前页面的控制台
